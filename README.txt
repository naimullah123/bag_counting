# Sack Counting using YOLOv8 and Computer Vision

## Overview
This project detects and counts sacks from warehouse video using YOLOv8 object detection and tracking.

## Features
- Real-time sack detection
- Sack counting based on line crossing
- Custom-trained YOLOv8 model
- Accurate counting for industrial scenarios

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Supervision Library

## Project Structure
- bag_counter.py – main counting script
- train_sack_model.py – model training script
- model/best.pt – trained sack detection model
